## 1. 游戏概述

### 1.1 故事背景

1640年，大西洋贸易时代的黎明。香料、蔗糖、白银、棉花——财富从新大陆滚滚而来，欧洲的商业家族们在这场财富盛宴中各施手段。玩家选择一个欧洲商人家族，在三角贸易的浪潮中建立贸易帝国，经历启蒙运动的思想革命，最终迎接蒸汽与铁的工业时代，书写两个世纪的商业传奇。

### 1.2 游戏时间线

| 章节 | 时代名称 | 年份 | 关键历史事件 | 主导经济 |
|------|---------|------|------------|---------|
| 第一章 | 帆影大洋 | 1640-1679 | 英荷战争、贸易公司崛起 | 三角贸易起步 |
| 第二章 | 商战烽火 | 1680-1719 | 殖民地争夺、海盗黄金年代 | 殖民垄断竞争 |
| 第三章 | 启蒙浪潮 | 1720-1759 | 南海泡沫、七年战争前夕 | 金融资本扩张 |
| 第四章 | 革命风云 | 1760-1799 | 美国独立、法国大革命 | 政治重构与机遇 |
| 第五章 | 蒸汽铁幕 | 1800-1840 | 拿破仑战争、第一次工业革命 | 工业资本霸权 |

- **起始**: 1640年
- **终止**: 1840年
- **总计**: 200回合（1回合 = 1年）

### 1.3 核心玩法

- **六大数值经营**: 财富、启蒙、产能、声望、航权、殖民度——每种数值独立生产与消费，通过数值接口影响彼此
- **三角贸易**: 欧洲→非洲→美洲→欧洲的商品循环，道德选择与利益权衡
- **殖民扩张**: 在世界地图上开拓殖民地，控制资源地块，建立贸易据点
- **海军博弈**: 战舰护航、制海权争夺、私掠许可
- **启蒙科技树**: 45项科技，覆盖航海、金融、工业、政治四大分支
- **工业革命**: 蒸汽时代解锁全新生产链，颠覆原有经济结构
- **家族传承**: 7个岗位、6维属性、最多14名成员、联姻外交
- **AI同机制**: 所有 AI 家族使用与玩家完全相同的数值系统
- **历史事件**: 50个固定事件 + 35个随机事件，真实历史节点抉择
- **四种胜利**: 财富霸权、工业称雄、殖民帝国、文化传承

### 1.4 核心战略闭环

本作的核心玩法不是单纯堆叠六大数值，而是在世界地图上经营一张会被竞争者、海军、殖民地和道德选择持续扰动的商业网络。

```
开辟贸易路线
  → 获得财富与战略资源
  → 建造舰队 / 扩张殖民地 / 投资工业
  → 控制关键海域与港口
  → 提高路线安全、压制对手、扩大殖民控制
  → 解锁更高价值路线与工业生产链
```

玩家每年的关键问题应当是：

- **贸易网络**: 选择开辟、升级或保护哪些路线，是否追求完整三角贸易。
- **局部制海权**: 将舰队部署到哪些海域，保护己方航线还是封锁/私掠对手。
- **殖民控制**: 在哪些资源地块推进探索、贸易站、殖民地和稳定控制。
- **道德代价**: 参与奴隶贸易、军事镇压、废奴改革或外交让步会改变收益、抵抗、声望和外交格局。

六大数值仍是系统接口，但它们服务于上述战略闭环：地图状态产生收益和风险，收益再转化为新的地图行动能力。

## 2. 核心循环与回合结构

### 2.1 回合 = 1年

每个回合代表一年，流程如下：

```
回合开始
  │
  ├─ 阶段0: 通胀推进（影响本年价格指数）
  ├─ 阶段1: 数值生产结算
  │    ├─ 财富产出（贸易路线 + 殖民地收入 + 家族产业）
  │    ├─ 启蒙产出（学术机构 + 学者成员 + 文化城市）
  │    ├─ 产能产出（工坊/工厂 + 殖民地原材料加成）
  │    ├─ 声望产出（胜利/成就 + 文化建筑 + 事件奖励）
  │    ├─ 航权产出（舰队规模 + 港口控制 + 海战胜利）
  │    └─ 殖民度产出（探险家 + 移民点 + 贸易站数量）
  ├─ 阶段2: 支出结算
  │    ├─ 工人/水手工资
  │    ├─ 舰队维护费
  │    ├─ 殖民地管理费
  │    └─ 家族运营费
  ├─ 阶段3: 贸易路线结算（三角贸易/直贸路线产出）
  ├─ 阶段4: 工业生产推进（产能消耗 → 建造/升级完成）
  ├─ 阶段5: 科技研发推进（启蒙点消耗 → 进度推进）
  ├─ 阶段6: 事件检查（固定 + 随机，可并发）
  ├─ 阶段7: 殖民地结算（控制度变化 + 抵抗运动 + 原材料产出）
  ├─ 阶段8: 外交结算（条约推进 + 声望自然衰减 + AI态度更新）
  ├─ 阶段9: 海军结算（舰队位置 + 制海权计算 + 私掠收益）
  ├─ 阶段10: AI势力行动（所有AI执行与玩家相同的决策逻辑）
  ├─ 阶段11: 胜利点结算（四类胜利条件进度）
  ├─ 阶段12: 家族成员培养推进
  └─ 阶段13: 推进年份 → 刷新 AP
```

### 2.2 玩家行动阶段

结算完成后进入行动阶段（phase = "action"），玩家自由分配行动点，直到"结束回合"。

## 3. 六大核心数值体系

> **设计原则**: 六大数值相互独立生产与消费，系统间只通过「数值接口」传递影响，不存在直接耦合。

### 3.1 数值总览

| 数值 | 图标 | 性质 | 核心用途 |
|------|------|------|---------|
| 财富 (Wealth) | 💰 | 主要货币（杜卡特） | 一切建造、外交、行动的基础消耗 |
| 启蒙 (Enlightenment) | 🔬 | 研究积累点 | 驱动科技树，解锁新时代能力 |
| 产能 (Industry) | ⚙️ | 生产力储量 | 建造船只、工厂、基础设施 |
| 声望 (Prestige) | 👑 | 政治社会资本 | 外交行动、政策解锁、联姻议价 |
| 航权 (Maritime) | ⚓ | 海洋控制力 | 贸易路线保护、海战、私掠许可 |
| 殖民度 (Colonial) | 🌍 | 海外扩张能力 | 探索新领土、建立殖民地、管理控制 |

### 3.2 数值生产来源

#### 💰 财富产出（每年）

| 来源 | 公式 | 说明 |
|------|------|------|
| 直贸路线 | 路线等级 × 15 × 通胀 | 每条贸易路线独立结算 |
| 三角贸易收益 | 见§6 | 特殊高收益贸易环路 |
| 殖民地税收 | 控制度 × 0.8 × 通胀 | 每个殖民地独立 |
| 港口码头 | 码头等级 × 20 × 通胀 | 每港口独立 |
| 家族产业 | 产业类型固定值 × 通胀 | 见§9 |
| 黄金对冲收益 | 囤积白银 × 0.3% | 抗通胀工具 |
| 航权被动收益 | 航权值 × 0.5 | 每点航权产生0.5财富/年 |

#### 🔬 启蒙产出（每年）

| 地理大发现事件 | +20~+40（一次性）| 随机事件 |

### 3.3 数值接口（模块间互动规则）

> 所有跨模块影响必须通过「数值接口」或「地图状态接口」表达，禁止系统直接互相改写内部状态。

#### 3.3.1 数值接口

| 数值 | 影响目标 | 接口公式 |
|------|---------|---------|
| 启蒙 | 解锁科技 → 其他数值产出加成 | 科技效果以 `+X%产能` 等数值形式表达 |
| 产能 | 建造成本 | 每点产能 = 可替代 2 财富用于建造 |
| 声望 | 外交财富折扣 | 每10声望 → 外交行动财富成本 -5% |
| 航权 | 财富被动产出 | 每1航权 → +0.5财富/年 |
| 殖民度 | 探索与殖民行动 | 支付殖民度 → 解锁探索、贸易站、殖民地推进 |
| 财富 | 声望 | 每年豪华消费（主动选项） → 声望 +X |
| 声望 | 联姻议价 | 声望越高 → 联姻对象家族属性越优质 |

#### 3.3.2 地图状态接口

地图不是展示层，而是核心结算层。贸易、殖民、海军系统先在地图上形成局部状态，再汇总为年度数值。

| 地图状态 | 影响目标 | 说明 |
|----------|---------|------|
| 贸易路线安全度 | 财富产出 | 路线经过的海域越安全，损耗越低 |
| 局部制海权 | 贸易、封锁、私掠 | 由舰队位置、任务和舰队航权共同决定 |
| 地块控制度 | 财富、产能、殖民度 | 资源地块按控制度产生收益和原材料输入 |
| 抵抗强度 | 控制度、声望、事件 | 压榨与镇压会提高抵抗，改革与让步会降低抵抗 |
| 港口等级 | 贸易、舰队、航线容量 | 港口决定路线起终点、舰队驻扎和补给能力 |

年度结算顺序：

```
地图状态更新
  → 计算路线安全 / 制海权 / 地块控制 / 抵抗
  → 汇总为财富、航权、殖民度、产能等数值变化
  → 玩家用数值支付下一轮地图行动
```

## 5. 经济系统

### 5.1 通胀模型

通胀因子基础值为1.0，随时间和事件变化：

```
通胀因子(t) = 1.0
  + 时代推进累加（每章+0.08）
  + 白银流入事件修正（美洲白银 → 通胀+0.15/次）
  + 战争状态（+0.05/年）
  - 贸易繁荣修正（-0.03/年，需贸易路线 ≥ 5）
  
范围: 1.0 ~ 3.5（钳位）
```

### 5.2 财富收支总结算

```
年度净财富 =
  + 贸易路线收益
  + 三角贸易收益
  + 殖民地税收
  + 港口收益
  + 家族产业收益
  + 航权被动收益
  + 事件修正收益
  - 工人/水手工资
  - 舰队维护费（舰船数 × 50 × 通胀）
  - 殖民地管理费（殖民地数 × 200 × 通胀）
  - 家族成员薪酬
  - 贷款利息（若有）
  - 税收（地方领主税/国家税）
```

### 5.3 税率计算

```
有效税率 = 基础税率(10%)
  + 战争修正（战时 +8%）
  - 声望减税（声望每10点 -0.5%）
  - 内政总监减税（满配 -15%）
  + 殖民地抵制事件修正
  
范围: 3% ~ 40%（钳位）
```

### 5.4 白银储备系统

白银（从美洲殖民地开采）作为通胀对冲工具：

| 白银储量 | 对冲效果 | 说明 |
|---------|---------|------|
| 0-19 | 无 | 无法对冲 |
| 20-49 | 工资支出 -10% | 部分对冲劳动力成本 |
| 50-99 | 工资支出 -20% | — |

## 6. 三角贸易系统

### 6.1 概述

三角贸易是本游戏最核心的财富获取机制，也是最重要的道德选择系统。三个贸易节点形成封闭回路，玩家可以主动参与其中的一段、两段或完整三角。

贸易路线不是单纯的列表项，而是地图实体：每条路线有起点港口、终点港口、经过海域、安全度、护航状态、私掠威胁和道德立场。玩家的舰队部署和殖民政策会直接改变路线收益。

```
欧洲（制造品） ──出口──▶ 非洲（西海岸）
                              │
                          劳动力（奴隶/契约工）
                              │
                              ▼
                          美洲殖民地（种植园）
                              │
                          原材料（糖/棉/烟）
                              │
                              ▼
欧洲（高价值货物） ◀──进口──────────────
```

### 6.2 贸易品种类

| 商品 | 产地 | 售价（基础） | 用途 |
|------|------|------------|------|
| 制造品 | 欧洲工坊/工厂 | 15/单位 | 输出到非洲换取劳动力 |
| 蔗糖 | 加勒比殖民地 | 40/单位 | 最高价值原材料，运回欧洲 |
| 棉花 | 北美殖民地 | 25/单位 | 工业时代关键原材料 |
| 烟草 | 弗吉尼亚/巴西 | 30/单位 | 高需求奢侈品 |
| 白银 | 秘鲁/墨西哥 | 50/单位 | 通胀对冲储备 |
| 香料 | 东印度群岛 | 60/单位 | 超高价值，需特殊航线 |
| 奴隶/契约劳工 | 非洲西岸 | — | 提升殖民地产出+40% |

### 6.3 三角贸易运作机制

#### 6.3.1 路线结构

每条贸易路线包含：

| 字段 | 说明 |
|------|------|
| 起点港口 | 欧洲、非洲、美洲或特殊远洋港口 |
| 终点港口 | 目标贸易节点 |
| 经过海域 | 路线穿过的海格或海域区 |
| 路线等级 | 代表船队规模、商站成熟度和货运组织能力 |
| 安全度 | 由局部制海权、护航任务、私掠威胁和事件决定 |
| 道德立场 | 合法贸易、奴隶贸易、契约劳工、废奴贸易 |
| 收益状态 | 正常、受威胁、被封锁、被私掠、繁荣 |

#### 6.3.2 完整三角路线

```
单次循环收益 = 
  (非洲货品交换价值 × 航运效率)
  + (殖民地原材料产出 × 劳动力加成)
  + (欧洲市场售价 - 运输损耗)

完整三角加成: ×1.5 收益乘数（相比单段路线）
```

完整三角路线要求三段路线均处于可用状态。任意一段被封锁、严重私掠或港口失控，完整三角加成暂停，只按仍可运行的单段路线结算。

#### 6.3.3 道德成本系统

| 参与方式 | 财富收益 | 声望代价 | 说明 |
|---------|---------|---------|------|
| 合法商品贸易（不参与奴隶贸易）| ×1.0 | 无 | 基础路线 |
| 参与奴隶贸易 | ×1.8 | -5声望/年 | 声望低于0触发国际制裁 |
| 废奴立场（主动废奴） | ×0.7 | +8声望/年 | 解锁废奴运动事件链 |
| 契约劳工替代 | ×1.2 | -1声望/年 | 折中选择，需科技解锁 |

道德选择必须产生地图后果，而不只是收益乘数：

| 选择 | 地图后果 | 长期风险 |
|------|----------|----------|
| 参与奴隶贸易 | 美洲殖民地产出提高，抵抗强度上升 | 抵抗扩散、外交制裁、废奴事件压力 |
| 军事镇压 | 控制度短期上升 | 周边地块抵抗上升，声望下降 |
| 外交收买 | 抵抗下降，控制度小幅上升 | 财富成本高，扩张速度慢 |
| 改革让步 | 抵抗大幅下降，声望上升 | 税收和短期殖民收益下降 |
| 废奴政策 | 解锁废奴路线和外交收益 | 旧有奴隶贸易路线收益下降或重组 |

> **设计意图**: 奴隶贸易带来巨大财富，但玩家会在地图上承担殖民地不稳定、外交阵营变化和事件链反噬。道德选择应当改变长期战略环境，而非成为可随时切换的收益开关。

### 6.4 贸易路线安全

贸易路线安全由局部地图状态决定，而不是只看全局航权。

```
路线安全度 =
  己方控制海域比例
  + 护航舰队加成
  + 港口补给与要塞加成
  - 敌方私掠压力
  - 风暴/战争/事件风险
```

| 安全状态 | 条件示例 | 每年损耗 |
|----------|---------|---------|
| 完全安全 | 绝大多数路径由己方或盟友控制 | 0% |
| 基本安全 | 少量路径受威胁，有护航 | 5% |
| 存在风险 | 多个关键海格处于敌方影响下 | 15% |
| 海盗频繁 | 路线缺少护航，敌方私掠活跃 | 30% |

私掠许可不再是全局修正，而是指定目标路线或海域：

- 对经过目标海域的敌方路线施加损耗。
- 若目标海域有敌方护航舰队，可能触发海战。
- 私掠收益越高，外交恶化与声望风险越高。

### 9.5 联姻系统

联姻是本作新增的核心外交手段，连接家族系统与外交系统：

```
联姻流程:
1. 外交总监发起提案（消耗 2AP + 1200财富）
2. 声望检查：声望 < 40 → 成功率 -30%
3. 目标家族态度检查（好感度 ≥ 30 可发起）
4. 成功 → 对方家族成员加入（携带属性）+ 好感度 +40 + 声望 +8
5. 失败 → 好感度 -10，声望 -3
```

联姻收益除属性外，还包含：
- 每年额外 +5 好感度维持（防止外交关系恶化）
- 联姻对象家族的特定科技共享（双方均解锁一项随机科技）
- 子代成年后携带双方家族血统属性

### 10.6 地块控制机制

每个可争夺地块可以同时存在多方控制度。控制度不是单一归属开关，而是地图前线状态。

```
控制度变化/年 =
  + 殖民行动加成（主动行动）
  + 驻扎舰队加成（港口格：+3/年/艘）
  - 抵抗运动（-5~-15/年，见§11）
  - AI竞争（相邻AI殖民地或同格竞争）
  + 基础设施建设 → 控制度上限提升
```

| 控制度 | 效果 |
|--------|------|
| 0-29 | 探索区域，无收益 |
| 30-49 | 贸易站建立，收益×0.5 |
| 50-69 | 殖民地建立，正常收益 |
| 70-89 | 稳定殖民地，收益×1.2 |
| 90-100 | 完全控制，收益×1.5，抵抗运动几率减半 |

## 11. 殖民系统

### 11.1 殖民扩张流程

殖民扩张是一条地图前线推进链，而不是一次性购买殖民地。玩家先探索，再发现资源和风险，随后选择建立贸易站、殖民地或放弃。

```
1. 派遣探险队（2AP + 600财富 + 消耗殖民度10）
   → 探索未知格，发现资源、港口点、土著抵抗或事件
   
2. 建立贸易站（2AP + 500财富 + 消耗殖民度10）
   → 控制度达30%，开始获取收益×0.5，解锁周边探索
   
3. 建立殖民地（2AP + 1000财富 + 产能20 + 殖民度20）
   → 控制度达50%，正常收益，可建设基础设施与驻军
   
4. 升级殖民地基础设施（1AP + 500财富 + 产能20）
   → 提升控制度上限，增加收益乘数，降低抵抗波动
```

### 11.2 多方控制与殖民前线

每个殖民地块维护 `势力 → 控制度`。当多个势力在同一资源区扩张时，控制度会形成竞争。

| 状态 | 条件 | 结果 |
|------|------|------|
| 未探索 | 玩家地图不可见或仅有传闻 | 无法直接殖民 |
| 已探索 | 已发现地块类型，资源可能未完全识别 | 可派探险队深入 |
| 贸易站 | 任一势力控制度达30 | 低收益，提供补给和影响力 |
| 殖民地 | 任一势力控制度达50 | 正常收益，可建设和驻扎 |
| 稳定殖民地 | 控制度达70且抵抗较低 | 收益提高，抵抗事件减少 |
| 完全控制 | 控制度达90 | 可作为区域核心，向周边投射影响 |

AI 与玩家争夺同一地块时，不必立刻宣战；可以通过投资、驻舰、外交收买、私掠、封锁或事件削弱对方控制。

### 11.3 殖民地原材料与产能接口

殖民地资源通过「原材料转化产能」接口影响工业系统：

| 原材料 | 每控制格/年 | 转化产能 | 说明 |
|--------|----------|---------|------|
| 蔗糖/烟草 | 30 财富 | +2 产能 | 轻工业原料 |
| 棉花 | 20 财富 | +4 产能 | 纺织工业关键 |
| 木材 | — | +6 产能 | 直接用于建造 |
| 铁矿 | — | +8 产能 | 重工业关键 |
| 煤炭 | — | +12 产能 | 工业革命核心 |

### 11.4 抵抗运动系统

抵抗强度是每个殖民地块的持续状态，会影响控制度增长、事件触发、军费和声望。抵抗不是单次惩罚，而是对殖民政策的长期反馈。

| 触发条件 | 抵抗强度 | 控制度损失 |
|---------|---------|----------|
| 参与奴隶贸易 | 高 | -12/年 |
| 苛刻税率（>30%）| 中 | -8/年 |
| 长期军事镇压 | 中 | -6/年 |
| 外交收买 | 低 | -3/年 |
| 宗教冲突（事件触发）| 高 | -10/年 |

**镇压选项：**

| 方式 | AP | 财富 | 效果 |
|------|-----|------|------|
| 军事镇压 | 2 | 400 | 控制度 +10，声望 -5，次年抵抗 +3 |
| 外交收买 | 1 | 600 | 控制度 +5，抵抗 -8，声望 +2 |
| 改革让步 | 2 | 800 | 抵抗 -15，税率强制降至20% |
| 放弃殖民地 | 1 | — | 完全撤出，声望 +3 |

抵抗会扩散到相邻殖民地块，尤其在长期军事镇压、奴隶贸易路线繁荣、宗教冲突事件和敌对势力煽动时。改革让步和废奴政策可以降低扩散。

## 12. 海军系统

### 12.1 舰船种类

| 舰船 | 航权值 | 建造产能 | 建造财富 | 维护/年 | 战斗力 |
|------|--------|---------|---------|---------|--------|
| 单桅商船 | +1 | 20 | 300 | 30 | 低 |
| 武装商船 | +2 | 30 | 500 | 50 | 低-中 |

| 战列舰 | +8 | 120 | 3000 | 200 | 高 |
| 铁甲舰 | +15 | 200 | 6000 | 350 | 极高 |

### 12.2 舰队任务系统

舰队不作为逐格战棋单位操作。玩家通过 AP 给舰队分配年度任务，路径、覆盖海域和冲突检测由后台地图系统计算。

| 任务 | 主要作用 | 风险 |
|------|---------|------|
| 护航路线 | 提高指定贸易路线安全度 | 可能遭遇私掠或海战 |
| 巡逻海域 | 提高目标海域局部制海权 | 维护费高，收益间接 |
| 封锁港口 | 降低敌方港口收益和路线容量 | 高外交风险，易触发战争 |
| 私掠航线 | 攻击非盟友贸易路线，获取财富 | 声望下降，关系恶化 |
| 护送探险 | 提高探索成功率，降低远洋事件风险 | 舰队被占用，无法护航贸易 |
| 驻扎港口 | 提高港口防御、周边控制度和补给能力 | 覆盖范围有限 |

舰队任务互斥，同一舰队每年只能执行一类主要任务。玩家的核心选择是舰队资源如何在收益、防守、扩张和压制对手之间分配。

### 12.3 局部制海权系统

地图被划分为6个海域区：

| 海域 | 关键航线 | 制海权价值 |
|------|---------|----------|
| 北大西洋 | 欧洲↔北美 | +10财富/年 |
| 加勒比海 | 糖/烟草产区 | +15财富/年 |
| 南大西洋 | 非洲↔美洲 | +8财富/年 |
| 地中海 | 地中海贸易 | +12财富/年 |
| 北海 | 欧洲北部 | +6财富/年 |
| 印度洋西部 | 香料路线 | +20财富/年 |

```
局部制海权 =
  舰队航权
  × 任务系数
  × 港口补给系数
  ÷ 距离衰减

海域控制率 = 该海域己方影响力 / 所有势力影响力

若制海权 ≥ 60% → 该海域贸易垄断 → 每年额外收益全额获取
若制海权 < 30% → 贸易路线受威胁 → 财富损失15%
```

全局航权仍保留，用于代表海军总实力、科技门槛和被动收益；局部制海权决定具体路线安全、封锁成功率和私掠风险。

### 12.4 私掠系统

私掠许可是半合法的海盗行为，带来航权和财富收益，同时消耗声望：

| 参数 | 值 |
|------|-----|
| 颁发成本 | 1AP + 100财富 |
| 每年私掠收益 | 随机 200~800 财富 |
| 目标 | 指定非盟友贸易路线或海域 |
| 声望代价 | -2声望/年（持续颁发时）|
| 外交风险 | 目标势力好感度 -10/年 |

私掠会在地图上产生可见威胁。若目标路线有护航舰队，可能触发海战；若目标势力制海权强，私掠收益降低且舰队损失风险提高。

## 14. AI 势力

### 14.1 设计原则

**所有 AI 势力使用与玩家完全相同的数值系统。** AI没有隐藏的特殊公式，它们同样生产六大数值、同样消费AP、同样受通胀影响。AI的差异仅来自初始属性、性格权重和家族加成。

### 14.2 AI家族列表（对手势力）

| 势力 | 母国 | 性格类型 | 擅长数值 | 弱点 |
|------|------|---------|---------|------|
| 范·奥兰治家族 | 荷兰 | 贸易扩张型 | 财富、航权 | 声望 |
| 科尔贝尔家族 | 法兰西 | 政治外交型 | 声望、启蒙 | 产能 |
| 格雷沙姆家族 | 英格兰 | 海洋霸权型 | 航权、殖民度 | 外交 |
| 博尔吉亚后裔 | 西班牙 | 殖民掠夺型 | 殖民度、财富 | 启蒙 |
| 富格尔家族 | 神圣罗马 | 金融工业型 | 产能、财富 | 航权 |
| 托马索家族 | 威尼斯 | 均衡精英型 | 均衡 | 无明显弱点 |

### 14.3 AI 决策框架

每年AI执行以下优先级逻辑（与玩家系统相同的数值驱动），但决策对象必须落到地图目标上：

```
1. 检查六大数值短缺（任一数值 < 阈值 → 优先补足）
2. 按性格权重分配AP：
   贸易型 → 贸易路线 > 殖民 > 科技 > 外交
   政治型 → 外交 > 声望建设 > 科技 > 贸易
   海洋型 → 海军建设 > 贸易 > 殖民 > 外交
3. 从地图中寻找目标：
   - 高价值未控制资源地
   - 安全度低但收益高的贸易路线
   - 己方核心航线上的敌方制海权缺口
   - 可封锁或私掠的敌方港口/航线
4. 评估威胁（邻近势力好感度 < -30、航线被私掠、殖民地被竞争 → 增加军事投入）
5. 执行最高优先级行动（扣AP，扣数值，产生地图状态变化）
6. 记录行动日志（与玩家可见的季报同格式）
```

AI 的行动必须可读：玩家应能通过地图图层看出 AI 为什么扩张某个资源地、巡逻某片海域、封锁某个港口或转向废奴/镇压政策。

## 15. 战斗系统

### 15.1 陆地战斗

陆地战斗仅在争夺殖民地控制权和欧洲领地时触发，使用简化数值对比。它不进入战术地图，而是作为殖民控制度冲突的结算事件。

触发方式：

- 玩家或 AI 对同一殖民地块投入军事镇压或驻军争夺。
- 殖民地抵抗强度过高并爆发武装冲突。
- 战争状态下，一方尝试夺取对方殖民地或欧洲领地。

```
战斗力 = 驻军数量 × 装备系数 × 地形系数 × 家族首领加成
胜负 = 攻方战斗力 vs 守方战斗力（+地形防御加成15-30%）
```

**战斗结果影响（纯数值接口）：**
- 胜利：控制度 +20，声望 +8，对方声望 -5
- 失败：消耗额外财富（抚恤），声望 -5

### 15.2 海战

海战使用舰队航权对比，决定局部制海权归属和贸易路线安全。海战由舰队任务冲突触发，而不是由玩家逐格追击触发。

触发方式：

- 护航舰队与私掠舰队在同一路线或海域相遇。
- 封锁港口时遭遇港口驻防舰队。
- 两个敌对舰队巡逻同一关键海域。
- 战争状态下舰队进入对方强控制海域。

```
海战胜负 = 攻方舰队航权总值 vs 守方舰队航权总值
  × 技术系数（双方科技差异）
  × 海将系数（航运总监属性）

胜利：敌方舰船损失30%航权；航权 +10；财富缴获
失败：己方舰船损失25%航权；贸易路线威胁期3年
```

海战结果必须回写地图状态：胜者提高目标海域制海权，败者的路线安全度下降，相关港口、封锁或私掠任务可能中断。

## 17. 家族选择（文明风格）

### 17.1 设计原则

玩家在开局选择家族，不同家族代表不同的历史原型和游戏策略。差异体现在：
- 初始数值加成（永久被动）
- 特殊能力（独特机制，每家族1~2个）
- 初始成员属性偏向
- 起始地理位置

## 18. 胜利条件

### 18.1 四种胜利路线（相对胜利制）

游戏在1840年结束，或任一势力提前达成关键里程碑后进入最终结算。胜利基于综合分值，但四个维度分别有「称霸门槛」。

#### 💰 财富霸权胜利

**触发条件**: 财富积累总量 ≥ 全部AI势力总和 × 1.5

| 评分项 | 权重 |
|--------|------|
| 年度财富净收益 | 40% |
| 贸易路线总数 | 20% |
| 商业据点数量 | 20% |
| 通货对冲能力（白银储量）| 20% |

#### ⚙️ 工业称雄胜利

**触发条件**: 产能产出 ≥ 所有AI之和，且拥有蒸汽工厂 ≥ 5座

| 评分项 | 权重 |
|--------|------|
| 年度产能产出 | 35% |
| 工厂/工坊数量 | 25% |
| 科技研究完成度 | 25% |
| 资源地块控制数 | 15% |

#### 🌍 殖民帝国胜利

**触发条件**: 殖民地数量 ≥ 8 且总控制度 ≥ 600

| 评分项 | 权重 |
|--------|------|
| 殖民地总控制度 | 30% |
| 殖民地资源多样性 | 20% |
| 殖民地覆盖大陆数 | 30% |
| 抵抗运动平息度 | 20% |

#### 👑 文化传承胜利

**触发条件**: 声望 ≥ 所有AI之和，且启蒙科技完成度 ≥ 75%

| 评分项 | 权重 |
|--------|------|
| 声望总量 | 35% |
| 外交盟约数量 | 20% |
| 启蒙科技完成度 | 25% |
| 联姻网络规模 | 20% |

### 18.2 失败条件

| 失败类型 | 触发条件 |
|---------|---------|
| 家族破产 | 财富 < -8000 且无可变卖资产 |
| 家族覆灭 | 在岗成员全部死亡/被俘 且无继承人 |
| 殖民地全失 | 所有殖民地控制度归零 且声望 < 10 |

### 19.2 地图视觉设计

- **六边形色块**: 每种地块类型有明确色彩编码

- 重大事件回顾
- 各AI势力动向（排行榜）
- 科技研究进度
- 胜利条件进度条
